# Projects

This document provides a quick map of the major projects included in this repository.

## Core Project

### CocoWeb Security Playground

- Path: [`漏洞靶场/`](./漏洞靶场)
- Type: Core project
- Stack: PHP, MySQL, HTML, CSS
- Status: Most complete / best entry point
- Summary:
  - A local web security playground with multiple vulnerability labs
  - Covers reflected XSS, stored XSS, SQL injection, SSRF, and CSRF
  - Supports Low / Medium / High security levels for teaching and comparison
